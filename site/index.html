This is a Game of Life simulation done using the WebGPU API. See the source code for details.

Game of Life is a simulation of a grid of cells. Each cell is alive or dead - initially at random.

A new generation is computed from the current one by applying a rule, which for each cell considers whether the current cell is alive or dead, and the number of alive neighboring cells.

A rule of the form B3/S1234 means that the a dead cell is born if it has 3 live neighbors, and a live cell will survive if it has 1, 2, 3 or 4 live neighbors.
